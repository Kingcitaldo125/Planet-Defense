# Planet-Defense
A simple 2D game done in [PyGame](pygame.org/docs)

## Running
First, install PyGame, if you have not done so using pip:
```
pip install -r requirements.txt
```

Run the game by invoking Python against `main.py`:
```
python main.py
```
